数值函数

1、小数位数处理

ROUND

ROUND(X, n)：对 X 进行四舍五入，保留 n 位小数，默认n为0

FORMAT

FORMAT(X, n)：对 X 进行四舍五入，保留 n 位小数，以##,###,###.###格式显示

FLOOR(X)

向下取整，返回小于这个数的最大整数

CEIL(X)

向上取整，返回大于这个数的最小整数

2、求余数、求幂、随机数

求余数MOD

MOD(X, Y)：求 X 除以 Y 的余数

求幂POW

POW(X, Y)：求X的Y次方

生成随机数RAND

2、字符串反转、拼接、具体替换

功能 函数 举例
反转 reverse reverse(‘123456’)：将字符串’123456’ 反转为 ‘654321’
拼接 pereat REPEAT(‘象飞田’, 3)：多个相同字符串拼接
concat CONCAT(‘马走日’, ‘象飞田’)：字符串拼接
concat_ws CONCAT_WS(‘-‘, ‘马走日’, ‘象飞田’)：指定分隔符拼接字符串
替换 replace REPLACE(‘赢赢赢士角炮巡河车赢’, ‘赢’, ‘’)：将所有’赢’替换为空
REPLACE(‘赢赢赢士角炮巡河车赢’, ‘赢’, ‘输’); 将所有’赢’替换为’输’

3、字符串截取

功能 函数 举例
指定截取 substr 或 substring SUBSTR(‘0123456789’, 3, 7)：从第3个字符开始截取，共截取7个字符，即’2345678’
左截取 LEFT LEFT(‘仙人指路，急进中兵’, 4); 从左边截取4位，即’仙人指路’
右截取 RIGHT RIGHT(‘仙人指路，急进中兵’, 4); 从右边截取4位，即’急进中军’

注意：mysql中，substring 与 substr 相同。

SUBSTR(str, n, m)：从 str 字符串的**第 n 个字符（注意：n不是下标）**往后截取 m 个字符，返回子串；m可省略，表示截取到末尾。

窗口函数

使用窗口函数时，必须加 over 关键字

语法：窗口函数 OVER(…)

分区 PARTITION BY

语法：窗口函数 OVER(PARTITION BY 字段，…)

PARTITION BY 字段的作用是按照指定字段的值对整张表的数据进行分区，OVER()中没有PARTITION BY时，整张表就是一个分区。
分区之后，在处理每行数据时，窗口函数是作用在该行数据关联的分区上，不再是整张表。

排名与排名函数

语法：排名函数 OVER( [PARTITION BY 字段] ORDER BY 字段 排序方式，…)

OVER() 中可以指定 ORDER BY 按照指定列对每个分区内的数据进行排序。
